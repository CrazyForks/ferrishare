Privacy Policy

Third parties

{{ global_app_name }} does not establish any connections to third-party servers. All assets are served locally.

Cookies and client data

If you are a normal user, {{ global_app_name }} does not store any cookies or data in your browser.

If you are the administrator of the site, logging into the site-wide administration panel will set a purely functional session-cookie.

Data collected during general use

In order to protect against Denial-of-service attacks your IP address is temporarily stored on the server and automatically deleted 24 hours after your last request.

Data collected during file upload

If you upload a file to {{ global_app_name }} the following information is recorded and stored in the server's database:

  • encrypted contents of the file
  • encrypted filename
  • time of upload
  • time of file expiry
  • your IP address

Your IP address is stored to ensure a fair and secure operation of the service. It is used to rate-limit malicious clients attempting to upload too many files at once.

The above data is deleted from the server once the file expires or is manually deleted.
